# Practice Workspace (PW) Feature Documentation

CPKit Practice Workspace (PW) provides a resizable layout workspace for competitive programmers to solve, dry-run, and organize problems.

---

## 🛠️ Implemented Features

### 1. Collections & Bookmarks
- Organize problems by standard categories: Arrays, Graphs, DP, Strings, Greedy, and Math.
- Bookmark questions to a dedicated collection tab.

### 2. Editor Options & Configuration Console
- Adjust typography dimensions (12px, 14px, 16px, 18px), wrap settings, and spacing modes (Standard vs Compact layout spacing).

### 3. Dry-Run Console Overlay
- Enter custom input parameters and simulate execution runs to verify outputs against sample examples in real time.
