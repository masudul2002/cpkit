# Quick Reference Center (REF) Feature Documentation

CPKit Quick Reference Center (REF) provides an instant-lookup searchable panel for STL containers, bit manipulation, modular arithmetic, graph/tree/DP/greedy cheatsheets, copyable C++/Python/Java templates, and compiler optimization flags.

---

## 🛠️ Key Sections

### 1. STL Reference
- **Supported Containers**: `vector`, `array`, `deque`, `list`, `stack`, `queue`, `priority_queue`, `set`, `multiset`, `map`, `multimap`, `unordered_map`, `unordered_set`, `bitset`, `pair`, `tuple`, and policy-based data structures (`PBDS`).
- Contains syntax declarations, average and worst-case time/space complexities, copyable code snippets, and common mistakes warnings.

### 2. Complexity Scales Table
- Visually logs arithmetic complexity counts across input size dimensions ($N = 10, 20, 50, 100$) for curves:
  - $O(1)$, $O(\log N)$, $O(N)$, $O(N \log N)$, $O(N^2)$, $O(2^N)$, $O(N!)$.

### 3. Bit Manipulation & Math Guides
- **Bitwise Tricks**: Power of two checks, popcount counts, and set/toggle/clear bit offsets.
- **Mathematics**: Modular exponentiation, Fermat's Little Theorem modular inverse, primes check, and combinations bounds.

### 4. CP Templates & Compiler Guides
- **Languages**: C++, Python, Java, Go, Rust templates with copy buttons.
- **Compilers**: Optimization flags (-O3, -Ofast) and address boundary debuggers flags (-fsanitize=address).
