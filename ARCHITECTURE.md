# CPKit Architecture

CPKit is a Next.js App Router workspace utilizing tailwind and standard typescript styles.

## Directory Layout

- `src/app/` — Pages routing entrypoints (sitemaps, settings, practice).
- `src/components/` — Global UI atoms (buttons, cards, dialogs) and Shell Layout layouts (navbar, sidebar).
- `src/features/` — Feature slices containing features-specific components, custom hooks, and utility files.
- `public/` — Static assets vectors.
