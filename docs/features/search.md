# CPKit Global Search Feature Documentation

CPKit integrates a global fuzzy command palette search (`Ctrl + K` or `Ctrl + Shift + P`) indexing tools, templates, cheatsheets, page settings, and bookmarks.

---

## 🛠️ Key Segments

### 1. Fuzzy Matches
- Standard fuzzy match parser logs targets against search queries.
- Instant outcomes lists highlighted results dynamically.

### 2. Category Checklists
- Filter search indices by category: Tools, Algorithms, Reference sheets, and Workspaces.
- Show recent activities lists directly inside empty search fields.
