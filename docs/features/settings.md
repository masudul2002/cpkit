# CPKit Settings Feature Documentation

The Global Settings configurator dashboard (`/settings`) allows competitive programmers to fully align visual properties, layout spacing, compiler preferences, and configuration backups files.

---

## 🛠️ Key Segments

### 1. Appearance & Theme Styles
- **Theme**: Light Mode / Dark Mode switching.
- **Accent highlights**: Accent color options (Cobalt Blue, Meadow Emerald, Quartz Violet, Sunrise Amber).
- **Fonts Sizing**: Scalable dimensions increments from `12px` to `18px`.
- **UI Spacing Density**: Toggle layout spacing standard vs compact mode.

### 2. Workspace Compiler Preferences
- **Default Lang**: Select primary compiler templates parameters (C++ G++20, Python 3, Java 17, Rust 1.75).
- **Transition animations**: Toggle simple micro-animations parameters.

### 3. Settings Backups Exports
- **Export**: Backup settings config directly to a client local JSON format file.
- **Import/Reset**: Hard-clear all properties from local storage parameters.
