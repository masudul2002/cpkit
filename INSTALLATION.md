# CPKit Installation

## Prerequisites
- Node.js (Version 18.x or above)
- npm (Version 9.x or above)

## Setup Steps
1. Clone the repository:
   ```bash
   git clone https://github.com/masudul2002/cpkit.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
